# NotePad</br>
<h3>一丶项目简介</h3>
是一个简单的记事本工具，包括保存便签，查询便签等功能.</br>
<h3>二丶功能简述：</h3>
1.为原有的note加入时间戳</br>2.添加搜索功能</br>3.更改了输入界面的壁纸</br>
